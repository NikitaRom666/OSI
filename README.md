# Практичне заняття: Реалізація шаблону Singleton

## Опис

Цей проєкт демонструє різні способи реалізації шаблону Singleton у C#.

## Структура проєкту

### 1. Простий Singleton (SimpleSingleton.cs)
Базовий варіант Singleton без потокобезпечності.

### 2. Singleton з лінивою ініціалізацією (LazySingleton.cs)
Екземпляр створюється тільки при першому виклику GetInstance().

### 3. Потокобезпечний Singleton (ThreadSafeSingleton.cs)
Використовує lock для забезпечення потокобезпечності.

### 4. Реальні приклади використання

#### DatabaseConnection.cs
Демонструє використання Singleton для з'єднання з базою даних.

#### ConfigManager.cs
Демонструє використання Singleton для менеджера конфігурації.

## Запуск

```
dotnet run
```

## Результати

Програма демонструє:
- Роботу простого Singleton
- Ліниву ініціалізацію
- Потокобезпечність
- Реальні приклади використання
- Порівняння з іншими підходами
